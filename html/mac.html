Терминал MacOS

  • ls - расшифровывается как "List files in the directory", что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • cd - расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • mv - расшифровывается как "Make directory", что переводится как "создать папку".
  • mkdir - расфшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать или удалить файл/

Git

  • git add-Добавляет файлы в индекс, чтобы они были включены в следующий коммит.
  • git status-Проверяет состояние репозитория, показывая, какие файлы изменены, добавлены в индекс или не отслеживаются.
  • git commit-Записывает изменения, добавленные в индекс, в историю репозитория.
  • git rm-Удаляет файлы из репозитория, удаляя их из индекса и рабочей копии.
  • git branch-Создаёт, переименовывает и удаляет ветки.
  • git checkout-Переключается между ветками или возвращает файлы к предыдущей версии.
  • git pull-Скачивает изменения с удаленного репозитория и сливает их с локальной веткой.
  • git push-Отправляет локальные коммиты на удаленный репозиторий.
  • git clone-Клонирует репозиторий, создавая локальную копию на вашем компьютере.
